# Inventory Management System

## Description

A Laravel + Livewire application for tracking inventory, stock movements, and product catalog management.

## Overview

This is a minimal viable product (MVP) for an Inventory Management System built with Laravel and Livewire. It provides real-time inventory tracking, basic stock operations, and simple reporting capabilities.

## MVP Features

- **Product Catalog**: Manage products with SKU, categories, pricing, and current stock levels
- **Category Organization**: Group products into logical categories
- **Supplier Management**: Track supplier information for procurement
- **Stock Operations**:
    - Stock In (receiving inventory)
    - Stock Out (deductions/adjustments)
- **Inventory Tracking**: Real-time stock levels with audit trail
- **Low Stock Alerts**: Visual indicators for items below threshold
- **Dashboard**: At-a-glance view of inventory status

## Requirements

- PHP ^8.1
- Composer
- Node.js & NPM
- MySQL 8.0+ (or PostgreSQL)
- Laravel 10+
